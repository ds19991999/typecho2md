# LInux下使用V2ray客户端

## 服务端
```
bash <(curl -s -L https://git.io/v2ray.sh)
```

## 客户端
下载脚本：https://pan.creat.kim/share/9Cjuoxat
```
安装
sudo bash go.sh
启动
sudo systemctl start v2ray
停止
sudo systemctl stop v2ray
重启 
sudo systemctl restart v2ray
配置文件
/etc/v2ray/config.json
```
